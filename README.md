
# Todo-Tui

A Todo app designed with [ratatui.rs](https://ratatui.rs/)


## Development

Clone the Git repository.
```sh
  git clone https://github.com/Zelvios/todo-tui.git
  cd todo-tui
```

Run the project.
```sh
  cargo run
```
